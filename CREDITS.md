## About the Project

**Aureum** is an innovative programming language created by Luiz Antônio, which combines Python's friendly syntax with Rust's extreme performance, implementing advanced AI techniques such as BitNet b1.58 and Matryoshka Representations.

### Creator's Vision

Develop a language that democratizes access to advanced AI optimization techniques, making them accessible through simple and intuitive syntax, while maintaining industrial-level performance.

---

## Implemented Technologies

### BitNet b1.58 (Ternary Computing)
Pioneering implementation of ternary weights {-1, 0, 1} with 2-bit compression, completely eliminating floating-point multiplications.

**Result:** 5-20x faster, 16x less memory

### Matryoshka Operator
Native implementation of the adaptive scale operator `[::scale]`, enabling dynamic trade-off between latency and precision.

**Result:** 2-4x fewer CPU cycles

---

## Hybrid Architecture

- **Frontend:** Python + Lark (parsing and transpilation)
- **Backend:** Rust (high-performance kernel)
- **Innovation:** Automatic transpilation with compile-time optimizations
