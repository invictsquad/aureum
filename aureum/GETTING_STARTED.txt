┌──────────────────────────────────────────────────────────────────────┐
│ KEY CONCEPTS                                                         │
└──────────────────────────────────────────────────────────────────────┘

  🔹 BitNet b1.58 (Ternary Computing)
     
     Weights: {-1, 0, 1} in 2 bits
     
     ❌ Traditional:  result = input * fp32_weight  (~5 cycles)
     ✅ Aureum:       match weight {
                        1 => add input         (~1 cycle)
                       -1 => subtract input    (~1 cycle)
                        0 => skip              (0 cycles)
                      }
     
     Gain: 5-20x faster, 16x less memory

  🔹 Matryoshka Operator
     
     Syntax: tensor[::scale]
     
     Example:
       weights[::256]   → Process 25% (ultra-fast)
       weights[::512]   → Process 50% (balanced)
       weights[::1024]  → Process 100% (maximum precision)
     
     Gain: 2-4x fewer CPU cycles
