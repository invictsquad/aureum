┌──────────────────────────────────────────────────────────────────────┐
│ TECHNICAL DIFFERENTIATORS                                            │
└──────────────────────────────────────────────────────────────────────┘

  1. BitNet b1.58 (Ternary Computing)
     ├─ Weights: {-1, 0, 1} in 2 bits
     ├─ Zero FP32/FP16 multiplications
     ├─ Only integer additions/subtractions
     └─ Gain: 5-20x faster, 16x less memory

  2. Matryoshka Operator
     ├─ Syntax: tensor[::scale]
     ├─ Adaptive processing
     ├─ Latency vs precision trade-off
     └─ Gain: 2-4x fewer CPU cycles

  3. Automatic Transpilation
     ├─ .aur code (Python-style)
     ├─ → AST (Lark parser)
     ├─ → Optimized Rust
     └─ → Native binary (x86_64/ARM)

┌──────────────────────────────────────────────────────────────────────┐
│ USAGE EXAMPLE                                                        │
└──────────────────────────────────────────────────────────────────────┘

  Aureum Code (inferencia.aur):
  ┌────────────────────────────────────────────────────────────────┐
  │ def inference():                                               │
  │     input = tensor(shape=[1024], type=int16)                   │
  │     weights = tensor(shape=[1024], type=bit1.58)               │
  │     result = input * weights[::512]                            │
  └────────────────────────────────────────────────────────────────┘
                              ↓
                    [Aureum Transpiler]
                              ↓
  Generated Rust Code (inferencia.rs):
  ┌────────────────────────────────────────────────────────────────┐
  │ use aureum_kernel::{pack_ternary, bitnet_infer};              │
  │                                                                │
  │ pub fn inference() {                                           │
  │     let mut input: Vec<i32> = Vec::with_capacity(1024);       │
  │     let mut weights: Vec<i8> = Vec::with_capacity(1024);      │
  │                                                                │
  │     // ═══ BitNet b1.58 + Matryoshka OPTIMIZATION ═══        │
  │     // Compression: weights → 2 bits/weight (4x smaller)      │
  │     // Matryoshka scale: only 512 elements processed          │
  │     // Zero FP multiplications: only +/- integers             │
  │     let packed_weights = pack_ternary(&weights);              │
  │     let result = bitnet_infer(&input, &packed_weights,        │
  │                                512);                           │
  │     // ═══════════════════════════════════════════            │
  │ }                                                              │
  └────────────────────────────────────────────────────────────────┘
