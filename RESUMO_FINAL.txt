                          5. HOW TO USE THE PROJECT
═══════════════════════════════════════════════════════════════════════════════

1. INSTALL DEPENDENCIES
   ─────────────────────
   pip install -r requirements.txt

2. COMPILE RUST KERNEL
   ────────────────────
   cd backend
   cargo build --release

3. TEST MATHEMATICAL TRUTH
   ──────────────────────────
   python test_verdade_matematica.py

4. VIEW COMPLETE TUTORIAL
   ─────────────────────
   cat examples/tutorial.aur

5. COMPILE TUTORIAL
   ─────────────────
   python frontend/aureum_compiler.py examples/tutorial.aur

6. USE INTERACTIVE REPL
   ────────────────────
   python main.py --shell
   >>> input = tensor(shape=[1024], type=int16)
   >>> weights = tensor(shape=[1024], type=bit1.58)
   >>> result = input * weights[::512]

7. RUN ALL TESTS
   ────────────────────────
   cd backend && cargo test --release
   cd .. && python test_compiler.py
   python frontend/aureum_ffi.py
   python test_verdade_matematica.py
